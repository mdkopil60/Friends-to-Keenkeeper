.............................KeenKeeper.................................

Description:

KeenKeeper is a React-based web app to help users manage and maintain their friendships.
Users can track interactions, set goals, and view friendship analytics.

---
Features :

View all friends with details
Add interactions (Call, Text, Video)
Timeline with filter option
Analytics with Pie Chart
Toast notifications
Fully responsive design

---

Technologies:

React.js
React Router DOM
Tailwind CSS + DaisyUI
Recharts
React Hot Toast
